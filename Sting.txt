Sting Programming Language
    - a beginner friendly interpreter based on Java

syntax:
    Comment - # This is a comment
    print statement - show()
    pipe | - statement/syntax seperator
    conditional statement - if(condition): values | elseif(condition): values | else: values ||
    while loop - loop(condition): values ||
    comma , - concatenation of variables and literals inside show()
    user input - roi() (return of input function)

datatypes:
    num (Integer, Float, Double, Long)
    str (String, Character)
    bln (Boolean)

operations/arithmetic:
    + addition
    - subtraction
    * multiplication
    / division
    % modulo
    ^ exponent
    ( left parenthesis
    ) right parenthesis
    = equals, assignment

comparisons:
    < less than
    > greater than
    <= less than or equal to
    >= greater than or equal to
    == is equals to
    != not equals to
    and &&
    or ||
    not !
